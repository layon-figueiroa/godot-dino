**Godot Dino** é um clone do clássico **jogo do dinossauro do Google Chrome**, desenvolvido com **Godot Engine**.  
O objetivo do jogo é sobreviver o máximo possível desviando dos obstáculos que aparecem no caminho.

Este projeto foi criado como prática de desenvolvimento de jogos utilizando **Godot** e **GDScript**.

---

# 🎮 Gameplay

O jogador controla o dinossauro e precisa evitar obstáculos que aparecem no cenário.

### Controles

| Tecla | Ação |
|------|------|
| **Barra de Espaço** | Pular |
| **Seta para Baixo** | Abaixar |

Se o jogador **colidir com qualquer obstáculo**, o jogo termina (**Game Over**).

---

# 🧠 Mecânicas do Jogo

- Sistema de **pulo**
- Sistema de **abaixar**
- **Spawn contínuo de obstáculos**
- **Detecção de colisão**
- Sistema de **Game Over**
- Loop infinito de gameplay

# 🛠️ Tecnologias Utilizadas
